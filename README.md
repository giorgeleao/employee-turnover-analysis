# Employee Turnover Analysis

Projeto de análise de rotatividade de funcionários utilizando Python e técnicas de People Analytics.

## Objetivo

Identificar padrões de desligamento de colaboradores através de análise exploratória de dados e visualizações.

## Tecnologias

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Estrutura

├── notebooks
│ └── turnover_analysis.ipynb
│
├── data
├── images
└── README.md

## Análises realizadas

- Distribuição de funcionários
- Rotatividade por departamento
- Tabelas de contingência
- Visualização de dados
